# Jupyter examples

## Preparation

1. Install Deno, e.g. on macOS:

```
brew install deno
```

2. Install Jupyter

```
pip3 install jupyterlab
```

3. Install Deno kernel

```
deno jupyter --unstable --install
```

4. Run Jupyter

```
jupyter lab
```

## Run examples

Then you can open in Jupyter any of the examples in this directory.
