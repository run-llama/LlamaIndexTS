## Introduction

### What are agents?

[Agents](./modules/agents/index.mdx) are LLM-powered assistants that can reason, use external tools, and take actions to accomplish tasks such as research, data extraction, and automation. 
LlamaIndex.TS provides foundational building blocks for creating and orchestrating these agents.

### What are workflows?

[Workflows](./understanding/workflows/index.mdx) are multi-step, event-driven processes that combine agents, data connectors, and other tools to solve complex problems. 
With LlamaIndex.TS you can chain together retrieval, generation, and tool-calling steps and then deploy the entire pipeline as a microservice.

### What is context engineering?

LLMs come pre-trained on vast public corpora, but not on **your** private or domain-specific data. 
Context engineering bridges that gap by injecting the right pieces of your data into the LLM prompt at the right time. 
The most popular example is [Retrieval-Augmented Generation (RAG)](./getting_started/concepts.mdx), but the same idea powers agent memory, evaluation, extraction, summarisation, and more.

LlamaIndex.TS gives you:

- **Data connectors** to ingest from APIs, files, SQL, and dozens more sources.
- **Indexes & retrievers** to store and retrieve your data for LLM consumption.
- **Agents and Engines** to query and use chat+reasoning interfaces over your data.
- **Workflows** for fine-grained orchestration of your data and LLM-powered agents.
- **Observability** integrations so you can iterate with confidence.

Alternatively, you can add `llamaindex` to your project and create an `index.ts` file to index your data.

Set your OpenAI key (or any other provider supported by [llamaindex](https://www.npmjs.com/package/llamaindex)):

```bash
export OPENAI_API_KEY=your-api-key
npm install llamaindex
```

Create an `index.ts` to index your data (in this case, a directory of documents in a folder called `data`):

```ts
import { SimpleDirectoryReader, VectorStoreIndex } from "llamaindex";

async function main() {
  const documents = await new SimpleDirectoryReader("data").loadData();
  const index = await VectorStoreIndex.fromDocuments(documents);
  const queryEngine = index.asQueryEngine();
  const response = await queryEngine.query("Some question about the data");
  console.log(response);
}

void main().catch(console.error);
```
